Аннотация

В рамках практики студент осваивает Git, Markdown и разработку HTML/CSS сайта.

О проекте

Проектная практика направлена на развитие навыков работы с Git и веб-разработкой.

  • Подготовка Markdown-документации.
  • Разработка HTML/CSS сайта.
  • Подготовка отчёта по практике.
